## Descrição

Projeto para aplicação de quadros de Kanban.

## Instalação

```bash
$ npm install
```

## Rodar a Aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Docmentação da API com Swagger

Após rodar o comando acima, acesse:

```bash
$ http://localhost:3000/api
```

## Database

Foi utilizado SQLite para o projeto.
